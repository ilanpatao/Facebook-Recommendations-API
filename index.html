  Facebook Recommendations API

This is a simple script designed to pull reviews and recommendations from Facebook for any business using the Page ID parameter.

Enter the Page ID of any location and hit the 'Get Reviews & Recommendations' button to see the results.

Please note to prevent abuse only 25 items will be returned.
